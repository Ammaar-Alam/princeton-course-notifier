About This App

This is a personal Course Sniper that queries the Princeton Student App API on a very tight interval so I can grab open seats fast.

  • Polling cadence: configurable down to every 1 second (this instance: {{ poll_interval }}s).
  • Re‑notify throttle: {{ renotify_secs }}s (sends another alert if the open count stays the same after this delay).
  • Compared to TigerSnatch’s default scheduler (~2–3 minutes), this runs much more frequently.

It’s built for my own use (and maybe a friend or two). Set your ntfy topic after login to receive push notifications on your phone.
